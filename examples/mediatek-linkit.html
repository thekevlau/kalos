MediaTek LinkIt ONE Position

This example shows you how to create an application that connects to a lightweight web server running on a MediaTek LinkIt ONE. From the server the position of the board is fetched and then displayed on a map from Google Maps.

Source Code

You can browse the source code for this example at the Evothings GitHub repository

The file index.html contains the HTML markup of the app and the buisness logic is placed in the app.js

The Arduino sketch can be found here.

What You Need

This example runs in Evothings Client on Android or iOS.

You need an iOS or Android device as well as a LinkIt ONE from MediaTek.

You also need a computer running Microsoft Windows or Ubuntu.

How To Get Up And Running

Follow these steps to get started with this example:

  1. Connect the WiFi and GPS antenna to the development board.
  2. Change the WIFI_AP, WIFI_PASSWORD and WIFI_AUTH to match your WiFi configuration.
  3. Compile and upload the changed sketch to your LinkIt ONE.
  4. Open a serial monitor and note the IP adress.
  5. Run the Evothings Workbench on your computer.
  6. Connect to the Workbench from the Evothings client app.
  7. Run the "MediaTek LinkIt ONE - Position" example from the Workbench window.
  8. Connect to your development board using the IP adress noted in previous step.
